The Machine Learning Web Application

Visit Application

Player recruitment is a crucial part of any sport, and soccer is no different. Teams are often looking for players with specific profiles - they might be looking to add a new player of a particular mould, or to replace an outgoing player. For this, the recruitment team might want to scour the market for a player who is similar to another player they know of. In order to aid this, I built a web application that uses an unsupervised machine learning algorithm (K-Means clustering) to process various metrics and cluster similar players. The application allows the user to choose a player and the desired age range for the suggestions, making it possible for anyone to quickly find 'the next XYZ'. I write about the entire process in further detail in my blog here.
